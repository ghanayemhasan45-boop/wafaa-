## 🎬 دورة حياة عملية: "إضافة منتج"

```
1. المستخدم ينقر "+"
   │
   ▼
2. onclick="addToCart(1)"
   │
   ▼
3. function addToCart(id) {
   ├─ const product = find(id)
   ├─ const existing = find in cart
   │
   ├─ if (existing) {
   │    existing.qty++  ← الكمية +1
   │ } else {
   │    cart.push({...product, qty:1})  ← إضافة جديد
   │ }
   │
   ├─ updateCartUI()  ← تحديث الرقم
   ├─ showToast(msg)  ← إشعار
   └─ toggleCart(true) ← فتح السلة
   │
   ▼
4. state.cart محدث
   │
   ▼
5. الواجهة محدثة
   │
   ▼
6. المستخدم يرى النتيجة
```

## 🛍️ دورة حياة السلة الكاملة

```
EMPTY CART
   │
   ├─ addToCart(1)  → cart = [{id:1, qty:1}]
   │
   ├─ addToCart(2)  → cart = [{id:1, qty:1}, {id:2, qty:1}]
   │
   ├─ addToCart(1)  → cart = [{id:1, qty:2}, {id:2, qty:1}]  (qty++)
   │
   ├─ updateCartQty(2, -1) → {qty: 0} → حذف
   │
   ├─ state.cart = [{id:1, qty:2}]
   │
   ├─ removeItem(1) → cart = []
   │
   └─ EMPTY CART
```

## 🔗 علاقات الدوال

```
init()
  ├─ renderFilters()
  │  └─ عرض أزرار الفئات
  │
  ├─ renderGrid()
  │  ├─ filter()
  │  ├─ sort()
  │  └─ عرض المنتجات
  │
  └─ setupEventListeners()
     ├─ searchInput.addEventListener()
     └─ sortSelect.addEventListener()

عند التفاعل:
  ├─ onclick="setFilter()" → renderFilters() + renderGrid()
  ├─ onclick="addToCart()" → updateCartUI() + showToast()
  ├─ onclick="toggleDashboard()" → initCharts() (أول مرة فقط)
  ├─ onclick="checkout()" → فتح WhatsApp
  └─ onclick="openImageModal()" → نافذة الصورة
```

```

   │

   ├─ Tailwind config

   ├─ State object
   ├─ DOM selections
   └─ init() function ← تشغيل البرنامج!
   │
   ▼
4. init() تشتغل:
   ├─ renderFilters() ← رسم الأزرار

   └─ setupEventListeners() ← تفعيل الأحداث
   │
   ▼
5. الموقع جاهز للاستخدام! ✅
```

## 📈 نمو البيانات في السلة

```
الحالة الأولى:
state.cart = []

بعد إضافة منتج واحد:
state.cart = [
  {id: 1, name: "منتج", qty: 1, ...}
]

بعد إضافة نفس المنتج مرة أخرى:
state.cart = [
  {id: 1, name: "منتج", qty: 2, ...}  ← qty++
]

بعد إضافة منتج آخر:
state.cart = [
  {id: 1, name: "منتج1", qty: 2, ...},
  {id: 2, name: "منتج2", qty: 1, ...}
]

الإجمالي = 900 + 180 = 1080 ج.م
```
