## 📄 شرح كل ملف

### 1️⃣ **index.html** - صفحة الويب الأساسية
**الدور:** يحتوي على البنية الأساسية للموقع

**المكونات الرئيسية:**
- **Header** (رأس الصفحة): شعار الموقع، أيقونة السلة، زر التحليلات
- **Hero Section**: صورة خلفية مع عنوان "فخامة لا تضاهى"
- **Dashboard Section**: رسوم بيانية (charts) تعرض:
  - توزيع المنتجات بشكل دائري (Doughnut Chart)
  - متوسط أسعار كل فئة (Bar Chart)
- **Filters & Search**: أزرار تصفية + حقل بحث + علبة ترتيب
- **Products Grid**: شبكة تعرض المنتجات (4 أعمدة على الويب، 2 على الجوال، 1 على الموبايل)
- **Cart Drawer**: درج جانبي يعرض منتجات السلة
- **Image Modal**: نافذة منبثقة لتكبير صور المنتجات

**المكتبات المستخدمة:**
- Tailwind CSS: تصميم سريع (CDN)
- Chart.js: رسوم بيانية (CDN)
- Google Fonts: خط Cairo العربي

## 🔧 الوظائف الرئيسية

### وظائف التهيئة والعرض:

| الوظيفة | الدور |
|--------|--------|
| `init()` | تشغيل البرنامج وتهيئة الفلاتر والمنتجات |
| `renderFilters()` | عرض أزرار التصفية بالفئات |
| `renderGrid()` | عرض شبكة المنتجات المفلترة والمرتبة |
| `toggleDashboard()` | إظهار/إخفاء رسوم البيانات |
| `initCharts()` | إنشاء الرسوم البيانية (Doughnut + Bar Charts) |

### وظائف البحث والتصفية:

| الوظيفة | الدور |
|--------|--------|
| `setFilter(id)` | تغيير الفئة المختارة |
| `setupEventListeners()` | ربط مستعمعات البحث والترتيب |
| `resetFilters()` | مسح جميع الفلاتر |

### وظائف سلة التسوق:

| الوظيفة | الدور |
|--------|--------|
| `addToCart(id)` | إضافة منتج للسلة |
| `updateCartQty(id, change)` | تعديل كمية المنتج (+1 أو -1) |
| `removeItem(id)` | حذف منتج من السلة |
| `updateCartUI()` | تحديث عرض السلة والكمية |
| `toggleCart(forceOpen)` | إظهار/إخفاء درج السلة |

### وظائف الدفع والإشعارات:

| الوظيفة | الدور |
|--------|--------|
| `checkout()` | إرسال طلب عبر WhatsApp |
| `showToast(msg)` | عرض إشعار مؤقت |

### وظائف الصور:

| الوظيفة | الدور |
|--------|--------|
| `openImageModal(src, alt)` | فتح صورة في نافذة منبثقة |
| `closeImageModal()` | إغلاق النافذة المنبثقة |

## 🎨 آلية العمل

```
1. عند فتح الموقع → init() تشتغل
   ↓
2. عرض الفلاتر (renderFilters) + المنتجات (renderGrid)
   ↓
3. المستخدم يختار فئة أو يبحث → renderGrid تظهر النتائج
   ↓
4. المستخدم يضيف لسلة → addToCart تضيفه ل state.cart
   ↓
5. الدفع → checkout يفتح WhatsApp برسالة الطلب
```
