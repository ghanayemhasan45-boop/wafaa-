# 📚 دليل سريع - Mira Gold Project

## 🎯 نظرة عامة

**Mira Gold** هو متجر إلكتروني متخصص ببيع المجوهرات بالعملة المصرية (جنيه مصري).

```
الموقع:   e-commerce لبيع المجوهرات
اللغة:    عربي (RTL - اليمين لليسار)
الدفع:    WhatsApp API
المنتجات: 115 مجوهرة في 6 فئات
```

## ⚡ الوظائف الرئيسية (الـ 17 دالة)

### التهيئة والعرض:
```
✅ init()              - بدء البرنامج
✅ renderFilters()     - عرض أزرار الفلاتر
✅ renderGrid()        - عرض قائمة المنتجات
✅ toggleDashboard()   - إظهار/إخفاء الرسوم البيانية
✅ initCharts()        - رسم الرسوم البيانية
```

### البحث والتصفية:
```
✅ setFilter(id)       - تحديث الفئة المختارة
✅ setupEventListeners() - تفعيل أحداث البحث والترتيب
✅ resetFilters()      - مسح جميع الفلاتر
```

### سلة التسوق:
```
✅ addToCart(id)       - إضافة منتج للسلة
✅ updateCartQty(id, change) - تغيير الكمية
✅ removeItem(id)      - حذف منتج من السلة
✅ updateCartUI()      - تحديث واجهة السلة
✅ toggleCart(forceOpen) - فتح/إغلاق الدرج
```

### الدفع والإشعارات:
```
✅ checkout()          - إرسال طلب عبر WhatsApp
✅ showToast(msg)      - عرض إشعار مؤقت
```

### الصور:
```
✅ openImageModal(src, alt)  - فتح صورة في نافذة
✅ closeImageModal()          - إغلاق النافذة
```

## 🎬 آلية العمل - خطوة بخطوة

### سيناريو 1: البحث عن منتج

```
1. المستخدم يكتب في حقل البحث: "ذهب"
   ↓
2. يُشغّل حدث "input" على searchInput
   ↓
3. state.searchQuery = "ذهب"
   ↓
4. renderGrid() تشتغل
   ↓
5. تصفية المنتجات:
   المنتجات اللي فيها "ذهب" في الاسم أو الوصف
   ↓
6. عرض النتائج فقط
```

### سيناريو 2: إضافة منتج للسلة

```
1. المستخدم ينقر على "+" على بطاقة منتج
   ↓
2. addToCart(1) تشتغل
   ↓
3. البحث عن المنتج في state.cart
   ↓
4. موجود؟ → زيادة qty
   غير موجود؟ → إضافة جديد
   ↓
5. updateCartUI() تحدث الواجهة
   ↓
6. عرض إشعار "تمت الإضافة"
   ↓
7. فتح درج السلة تلقائياً
```

### سيناريو 3: الدفع عبر WhatsApp

```
1. المستخدم ينقر "إتمام الشراء"
   ↓
2. checkout() تشتغل
   ↓
3. بناء الرسالة:
   "مرحباً ميرا جولد،
    أود طلب:
    - طقم الزهرة (2) - 900 ج.م
    - عقد اللؤلؤ (1) - 180 ج.م
    الإجمالي: 1080 ج.م"
   ↓
4. فتح WhatsApp
   ↓
5. إرسال الرسالة
```

## 🐛 استكشاف الأخطاء

### المشكلة: المنتجات لا تظهر
```
✓ تحقق من أن products[] ليست فارغة
✓ تأكد من loadScript قبل HTML
✓ افتح Console (F12) وتحقق من الأخطاء
```

### المشكلة: السلة لا تعمل
```
✓ تأكد من onclick="addToCart(1)"
✓ تحقق من state.cart في Console
✓ أعد تحميل الصفحة
```

### المشكلة: الرسوم لا تظهر
```
✓ انقر على "تحليلات"
✓ تأكد من أن Chart.js محمّلة
✓ تحقق من Canvas IDs
```

### المشكلة: WhatsApp لا يفتح
```
✓ تحقق من رقم الهاتف
✓ استخدم صيغة دولية (+201...)
✓ تأكد من encodeURIComponent()
```
